# Climate Raster Data (COG Format)
Place climate variable COG files here, organized by country and variable.

Example structure:
```
climate/
├── bhutan/
│   ├── maximum_temp/
│   │   ├── historical/
│   │   │   ├── annual/
│   │   │   │   └── max_temp_annual.cog
│   │   │   └── seasonal/
│   │   │       ├── max_temp_12_02.cog
│   │   │       └── [other seasons]
│   │   └── [scenarios]/
│   └── [other variables]/
└── [other countries]/
```
